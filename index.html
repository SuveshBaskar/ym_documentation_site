Response Functions

Send Text Message

Send a basic text message to the user.

app.sendTextMessage("message")

Send Quick Replies

Send a basic text message along with options to the user.

app.sendQuickReplies({ 
  title: `text`,
  options: [
    {
      title: "text",
      text: "text"
    },
    {
      title: "text",
      text: "text",
    }
  ]
});

Send Cards

Send Cards along with options to the user.

app.sendCards(
  {
    title: `title`,
    image: `image url`,
    text: `text/description`,
    actions: [
        {
            title: `title`,
            text: `text`
        }
    ]
  }
);
